# JosephMaza.com

Static personal brand website for Joe Maza.

## Local Preview

This site does not require a build step. From the project root, run:

```sh
python3 -m http.server 4173
```

Then open:

```txt
http://127.0.0.1:4173/
```

## Cloudflare Pages

Use the project root as the static output directory. No framework preset, build command, or backend is required.
